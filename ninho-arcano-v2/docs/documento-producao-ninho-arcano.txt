

Projeto: Ninho Arcano
Plataforma inicial: Android
Tipo: Jogo mobile de cuidado, mistério, evolução em tempo real e dragão pessoal
Stack recomendada: Expo + React Native + TypeScript
Objetivo: Gerar um projeto completo, organizado, moderno, fluido, visualmente épico e pronto para compilar APK


---

1. MISSÃO DA IA CONSTRUTORA

Você deve criar um aplicativo/jogo mobile completo chamado Ninho Arcano, pronto para rodar em Android e pronto para compilar APK.

O jogo deve seguir fielmente o conceito abaixo:

> Um universo caiu. Um ovo chegou até você.



O jogador recebe um Ovo Arcano Primordial vindo de um universo destruído chamado Vaelun. Esse ovo cria um vínculo com o aparelho do jogador. Cada aparelho só pode ter um ovo/dragão por vez. O jogador cuida do ovo em um Berço Arcano neutro, oferece orbes elementais baseadas no Wu Xing e acompanha mudanças sutis até o nascimento do dragão.

O jogo não deve parecer sistema, painel, lista ou CRUD. Deve parecer uma experiência viva, mágica, moderna, fluida, emocional e cinematográfica.

Regra máxima de design:

> Não mostrar sistema. Mostrar vida.




---

2. ESTILO E QUALIDADE ESPERADA

O app deve ser construído com foco em experiência visual e emocional.

Não criar interface quadrada, robótica, seca ou parecida com formulário.

O jogo deve ter:

abertura cinematográfica;

frases surgindo com animação;

fundo escuro e mágico;

partículas suaves;

transições fluidas;

elementos flutuantes;

orbes arrastáveis;

Berço Arcano com vida;

ovo pulsando;

luz de tochas tremulando;

runas acendendo;

reações visuais e textuais;

animações em todas as interações importantes;

linguagem poética;

tela principal visual, não cheia de menus.


Priorize beleza, fluidez e experiência. O app pode usar bibliotecas de animação e efeitos. Ainda assim, mantenha o código organizado e estável para Android.


---

3. STACK OBRIGATÓRIA RECOMENDADA

Criar projeto com:

Expo;

React Native;

TypeScript;

AsyncStorage para salvamento local;

React Navigation;

React Native Reanimated;

React Native Gesture Handler;

Expo Linear Gradient;

Expo Blur ou BlurView, se disponível;

Expo Haptics, se disponível;

Expo AV para sons opcionais;

Lottie ou animações por código, se necessário;

Preferencialmente Skia ou componentes animados para partículas/orbes se a IA conseguir implementar de forma estável.


O projeto deve estar pronto para rodar com:

npm install
npx expo start

E preparado para build Android via EAS:

npm install -g eas-cli
eas build -p android --profile preview

Também pode incluir alternativa para gerar APK local se possível, mas o caminho principal deve ser Expo/EAS.

5. LORE OFICIAL

5.1 Vaelun

O universo original dos dragões se chama Vaelun.

Em Vaelun, os dragões não eram lenda. Eles tinham ninhos, escolas, clãs, linhagens, guardiões, sociedades e tradições. Toda a vida dracônica era moldada pelas cinco forças do Wu Xing:

Madeira;

Fogo;

Terra;

Metal;

Água.


5.2 A Queda dos Ninhos

Vaelun foi destruído no evento chamado:

A Queda dos Ninhos

Quando Vaelun caiu, os últimos guardiões reuniram os Ovos Arcanos Primordiais e os enviaram através do vazio para este universo.

O jogador recebe um desses ovos.

Frase oficial:

> Você não escolheu um dragão. Você recebeu um sobrevivente de Vaelun.




---

6. REGRAS CENTRAIS DO JOGO

6.1 Um aparelho, um ovo

Regra absoluta:

> Cada aparelho só pode sustentar um único Ovo Arcano Primordial por vez.



Enquanto existir um ovo/dragão vinculado ao aparelho, nenhum outro ovo pode surgir.

A experiência deve deixar isso muito claro no início.

6.2 Travessia

A única forma de liberar o aparelho é realizar a Travessia.

Travessia significa mover o vínculo para outro aparelho. Não é cópia. Não é duplicação.

Frase oficial:

> Um vínculo não se copia. Um vínculo atravessa.



Nesta versão local/offline, criar a tela de Travessia como função futura simulada/explicativa. No futuro online, ela precisará transferir o vínculo via servidor.

6.3 Energia não acelera idade

As energias moldam o ovo e o Berço. Elas não aceleram o tempo.

Regra oficial:

> A energia molda. O tempo revela.



6.4 Tempo real

Regra de idade:

1 dia real = 1 ano dracônico

Dragões podem viver até 10.000 anos dracônicos.

Dragões só são adultos a partir de 1.000 anos dracônicos.

A Fase 1 vai apenas até Filhote II.

9. ACEITAR O VÍNCULO

Após a abertura, mostrar tela de vínculo.

A tela deve parecer ritual, não contrato.

Texto:

Este aparelho pode sustentar apenas um Ovo Arcano Primordial.

Ao aceitar o vínculo, este ovo passará a existir aqui.

Enquanto ele permanecer neste aparelho, nenhum outro ovo poderá surgir.

A única forma de liberar este aparelho é realizar a Travessia.

Botão principal:

> Aceitar o vínculo



Botão secundário:

> Ainda não



Ao aceitar:

gerar bondId único local;

marcar hasAcceptedBond = true;

criar game state inicial;

ir para Berço Arcano.

sem cristais grandes;

forma sagrada e frágil.


Ele deve transmitir:

11.2 Camadas do ovo

12. ORBES ELEMENTAIS — WU XING

As energias aparecem como orbes vivas flutuando.

Não usar lista. Não usar dropdown. Não usar botão seco.

O jogador deve tocar e arrastar as orbes.

12.1 Elementos

Madeira;

Fogo;

Terra;

Metal;

Água.


12.2 Visual das orbes

Elemento	Visual

Madeira	verde, orgânico, fios como raízes, brilho vivo
Fogo	âmbar/vermelho, brasas, calor ondulante
Terra	dourado/marrom, peso visual, poeira mineral
Metal	prata/cinza, brilho limpo, reflexo cortante
Água	azul profundo, líquido, névoa suave


12.3 Interação correta

Na tela principal, as orbes flutuam ao redor do ovo.

Fluxo:

Orbe flutua
↓
Jogador toca e segura
↓
Orbe aumenta brilho
↓
Jogador arrasta
↓
Se soltar sobre o ovo: oferecer ao ovo
↓
Se soltar sobre chão/base/parede do Berço: infundir no Berço
↓
Cena reage visualmente
↓
Mensagem poética aparece

Não perguntar em menu para onde vai. O gesto decide.

12.4 Zonas de drop

Criar duas zonas invisíveis:

eggDropZone;

cradleDropZone.


Se a orbe for solta no ovo, aplicar offerOrbToEgg(element).

Se for solta no Berço, aplicar infuseOrbToCradle(element).

chifres mais definidos;

cauda mais expressiva;

postura entre juventude e força;
